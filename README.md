# Organizer

An Electron-based desktop application for managing local workspaces and coordinating agentic coding tasks.

## Key Features
- Native macOS desktop experience
- Tight integration with local development workspaces
- Built-in support for agentic development workflows

## Installation
Go to the **[Releases](https://github.com/guocity/organizer/releases)** page to download the latest signed and notarized installer for macOS:
- **macOS Apple Silicon (ARM64)**: `Organizer-1.1.0-arm64.dmg` or `Organizer-1.1.0-arm64-mac.zip`
